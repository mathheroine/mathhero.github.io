标题语法

要想创建标题，只需在单词或短语前加井号“#”，接空格后写标题内容。其中井号“#”数量表示了标题的级别：
“#”：

level1

“###”：

level3

除了使用井号外，还可在文本下方添加任意数量“=”或“—”，分别表示一级标题与二级标题
level1

level2

段落语法

要想创建段落，需用一行或多行空格分隔文本（像latex）

I really like using markdown.

I think I’ll use it to format all of my documents from now on.

注意不要用空格（space）或制表符（tab）来缩进！

换行语法

在一行的末尾添加2个或更多空格，按回车便可换行：
This is my first line.
And this is the second line.
（感觉直接按回车也能换行，没什么区别）
为了兼容性，换行需要在末尾添加空格或者“$
$” 表示换行，不要用反斜杠换行。

强调语法

一般通过将文本设置为粗体或者斜体表示强调：
1.在单词或短语的前后添加2个星号或下划线来加粗
I just love bold
text.

I just love bold
text.

如果是在单词或短语中间部分加粗，则使用星号(为了兼容性，不用下划线)：
loveisbold

2.在单词或短语的前后添加2个星号或下划线使用斜体：
I just love bold
text.

I just love bold
text.

如果是在单词或短语中间部分加粗，则使用星号
Acatmeow

3.如果想同时加粗与斜体，则使用三个星号或下划线。

引用语法

创建块引用：在段落前加“>”

这是一个块引用

块引用内可以包含段落，在空白行使用“>”即可

这是一个块引用

用“>”换行后产生新段落

块引用也可以嵌套，在引用内使用“>>”即可

这是一个块引用

套娃

块引用内可以包含一些markdown格式的元素（并不是全部包含），例如：

这是一个块引用

  • Markdown
  • Markdown

everything is going according to plan.

列表语法

我们可以创建有序列表和无序列表

例如有序列表：在每个项目前加数字并紧跟句点（句点后接空格再写内容），数字不必按自然数顺序，但是必须以1开始

  1. 第一项

  2. 第二项

  3. 第一项

  4. 第二项

  5. 第三项

这里跟markdown官方文档里不同，官方文档中不论数字是否依自然数顺序，显示的序号都是自然数顺序的，可能和obsidian有关。为了兼容性，不用 1) 的形式创建列表。

至于无序列表，可以用破折号-，星号*或者加号+创建无序列表

  • 第一
  • 第二
  • 第一
  • 第二
  • 第一
  • 第二

进一步地，通过缩进表示列表的嵌套：

  • 第一
    1. 缩进
    2. 缩进
  • 第二

就是在编辑的时候调整破折号、星号或加号的位置，obsidian里会自动调整到合适的位置。
注意创建无序列表的时候不要混合使用这些符号，应使用同一个符号

注意如果想在保持列表连续性的同时，在列表中插入其他markdown元素（比如插入块引用时仍是以>开头），只需要把该元素缩进4个空格也就是一个制表符（实测按tab即可）：

  • 第一项

  • 第二项
    插入一个段落

  • 第三项

  • 第一项

    插入一个块引用

  • 第二项

特别地，在markdown文本中插入代码块通常是用4个空格即1个tab缩进，但是要想在列表中插入代码块，则应以8个空格即2个tab缩进(要用3个反引号`分别包裹代码头尾)：

  1. Open the file.
  2. Find the following code block on line 21:
  3. Update.

代码语法

如果在单词或短语中某一部分需要表示为代码，则可将其包裹在反引号中：
At the command prompt, type nano.

如果在句子中需要“打出”反引号，则可将句子包裹在双反引号中：
Use `code` in your markdown file.

创建代码块：缩进4个空格即1个tab：(注意用空行将代码块与文本分隔)

</html>

链接语法

要添加超链接，只需将想要显示的文本写在方括号，url写在后边括号里即可：

这是一个超链接：Markdown语法

超链接的title是指鼠标悬停在文本上显示的文字，要想显示title，可以把title写在url后面（空格分隔，title用双引号）：

这是一个超链接：[Markdown](https://markdown.com.cn “最好的markdown教程”)

(但是在obsidian里怎么不显示title啊？？？)

除了用文本显示超链接，还可以直接以网址形式显示超链接，用尖括号包裹网址或email即可：

这是一个超链接：https://markdown.com.cn
这是我的邮箱：18365408828@163.com

当然可以在链接中加入markdown格式：

粗体超链接，即在超链接外包裹双星号： I love supporting the EFF.

斜体超链接，在超链接外包裹星号：This is the Markdown Guide.

代码形式的链接，在方括号的文本外包裹反引号： See the section on code.
(好像没什么效果)

注意：为了兼容性，如果url中含有空格，请用%20表示空格：
例如https://www.example.com/my great page表示为：

自动网址链接

在许多markdown处理器中，例如obsidian，会将url自动转化为链接，即使不使用方括号，也会处理为链接：

www.example.com

若不希望如此，可以使用反引号包裹该网址（表示为代码）

www.example.com
